# AI Guest Review Analyzer (Frontend)

Frontend application for the AI Guest Review Analyzer project built using React, Vite, and Tailwind CSS.

## Tech Stack

* React 19
* Vite
* Tailwind CSS 4
* React Router DOM
* Lucide React
* React Hot Toast

# Running the Project

```bash
npm install
npm run dev
```

Vite will start the development server at:

http://localhost:5173

If the port is occupied, Vite automatically selects the next available port.
